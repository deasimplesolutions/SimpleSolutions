Chat Widget Demo

A floating AI chatbot powered by Claude. Click the chat button in the bottom-right to try it out.

↓
Quick Start

Setup in 3 Steps

  1. Open chatbot.js and replace "your-api-key-here" with your Anthropic API key.
  2. Add the CSS and JS files to any page (see embed code below).
  3. That's it — the widget will appear in the bottom-right corner.

Embed Code

<!-- Add inside <head> or before </body> -->
<link rel="stylesheet" href="chatbot.css" />
<script src="chatbot.js"></script>
Customization

Make It Yours

1. API Key

Open chatbot.js and set the variable at the top:

// Replace with your Anthropic API key
const ANTHROPIC_API_KEY = "sk-ant-...";

2. System Prompt

Edit SYSTEM_PROMPT in chatbot.js to control the chatbot's personality, knowledge, and tone:

const SYSTEM_PROMPT =
  "You are a helpful assistant for Simple Solutions.
   Answer questions about our services, pricing,
   and availability. Be friendly and concise.";

3. Business Name

Change BUSINESS_NAME in chatbot.js:

const BUSINESS_NAME = "Simple Solutions";

4. Accent Color

Open chatbot.css and change the CSS variable at the top:

:root {
  --chat-accent: #4f46e5;       /* main color */
  --chat-accent-hover: #4338ca; /* hover state */
}
Architecture

File Structure

your-website/
  ├── chatbot.css        # All widget styles
  ├── chatbot.js         # Widget logic + API calls
  └── chatbot-demo.html  # This demo page

The widget is fully self-contained — just the CSS and JS files. Drop them into any existing website and they'll work without conflicts.

Important

Security Note

This widget calls the Anthropic API directly from the browser, which means your API key is visible in the page source. For production use, you should route API calls through your own backend server to keep the key secret. The current setup is ideal for internal tools, prototypes, and demos.
